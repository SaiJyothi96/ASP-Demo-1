About Us

This Machinery Booking Management System was developed as a part of our project to streamline the process of booking agricultural machinery for farmers. The system provides a user-friendly platform for admins and farmers to manage and schedule machinery usage efficiently.

Technologies Used

  • Frontend: HTML, CSS, Bootstrap, JavaScript
  • Backend: Python (Flask Framework)
  • Database: MySQL

Key Features

  • Admin and Farmer Login
  • Real-Time Machinery Booking with Date & Time Filters
  • Booking Status Tracking (Booked, Returned, Payment Pending)
  • Automated Count of Machinery and Bookings on Admin Dashboard
  • Date-Time Based Search for Booking History
